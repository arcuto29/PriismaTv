# PriismaTv v2

> A premium personal streaming platform with Solo Leveling / Jin-Woo branding. Complete redesign from scratch — better than Netflix/Disney+.

## Tech Stack

- **Next.js 16** (App Router)
- **React 19** + TypeScript
- **Tailwind CSS v4**
- **Framer Motion** (smooth page transitions & micro-interactions)
- **Embla Carousel** (content row carousels)
- **Lucide React** (icons)
- **PWA** installable

## Features

- 180+ titles (movies, anime, TV shows)
- Cinematic hero section with auto-rotating featured content
- Premium card layouts with hover interactions
- Multiple streaming servers + season/episode selector
- Sub/dub support for anime
- Watchlist, Favorites, History, Continue Watching
- Chat widget, Watch Together rooms, YouTube & Twitch embeds
- Collections, Challenges, Calendar & Countdown timers
- Spin the Wheel random content picker
- Content Requests system
- Add content with TMDB auto poster fetch
- Owner password lock
- Load New Content, Edit, Export/Import
- Light/Dark mode with Jin-Woo backgrounds
- PWA installable, friend sharing via codes
- Fully responsive (mobile, tablet, desktop)

## Getting Started

```bash
pnpm install
pnpm dev
```

Connect this repo to [Vercel](https://vercel.com) and it will auto-deploy.
